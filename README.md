StandIn is a peer-to-peer platform to match individuals facing a time crunch with those who are willing to help with their errands or tasks. Full-stack web app built using Ruby on Rails and PostgreSQL.

## Development
* [Ruby 5.0.2](https://www.ruby-lang.org/en/)
* [Ruby on Rails](https://rubyonrails.org/), [PostgreSQL](https://www.postgresql.org/)
